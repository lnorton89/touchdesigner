# Plan 02-01: Agent Conversation MVP

## Goal

Build the first Agent COMP behavior on top of the proven Model Router path.

## Scope

- Add a pure-Python Agent extension with system prompt, message DAT input, history, output DAT writing, status channels, and clear/append helpers.
- Extend `ModelRouter.request(...)` with per-request provider, endpoint, model, timeout, and API-key-source overrides.
- Add unit tests for Agent history/message construction, output DAT mutation, clear behavior, and Router override propagation.
- Add a generated-demo Agent surface and startup smoke without destabilizing the existing bridge/Router smoke.

## Non-Goals

- Tool calling belongs to Phase 3.
- `.tox` packaging and dependency bootstrap remain Phase 4.
- Broad UI/custom-parameter polish is deferred until the operator family is packaged.
